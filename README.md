## Sobre o Projeto

O **Meu Bairro Conecta** é uma aplicação web desenvolvida como Prova de Conceito (PoC) para fortalecer a economia local. O sistema conecta moradores a prestadores de serviços e pequenos comerciantes do mesmo bairro, facilitando a busca por profissionais de confiança e fomentando o senso de comunidade.

### Funcionalidades Principais (MVP)
* **Listagem de Serviços:** Visualização de prestadores cadastrados com categoria e descrição.
* **Cadastro de Prestadores:** Formulário para novos profissionais divulgarem seus serviços.
* **Contato Direto:** Integração via link para WhatsApp do prestador.
* **Persistência de Dados:** Armazenamento das informações em banco de dados relacional.
